Build a semantic parsing based QA system using QANARY architecture and an end-to-end neural network based QA system over knowledge base. Also understand the current research trends and state of art systems in the field.

Content Overview

Introduction to Question Answerig

Overview

In this section of the tutorial we will give an overview of QA over knowledge graphs. This will include common source KGs, the datasets used to evaluate QA systems, the prominent types of approaches, and familiarizing the audience with the sub-tasks (like entity linking, predicate linking) and the terminology used in the community. We will then introduce the basic concepts of NLP needed for semantic parsing based system

Session 1 time

Semantic Parsing

QA using QANARY

In this session we will build the first QA system of the tutorial using QANARY architecture. While creating a QA system, we will aslo introduce different aspects of Semantic Parsing system and also explain how to build components for QANARY architecture

Session 2 time

Recent works

Introduction to Deep learning based QA

The sessions will introduce more complex forms of semantic parsing based systems. and discuss briefly some alternate mechanism of QA, like rule learning based system. We will then move to end-to-end QA system (Neural Network), introduce the basic concepts and the prerequisites needed for the final session (Implementing a simple Neural network based QA)

Session 3 time

End-to-End QA

Implement a Simple neural network based QA

In this session, we will give a practical introduction for the development and training of a simple question answering system using neural networks, focusing on the implementation of a learning-to-rank approach for QA in PyTorch. First, we will show how to write a basic neural model for ranking questions, how to prepare training data and how to train and evaluate the model. Then we will discuss possible improvements to the model, such as handling out-of-vocabulary words and better predicate representation.
